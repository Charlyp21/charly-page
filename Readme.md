Bienvenido a mi terminal personal. Soy estudiante de **Neurociencias en la UNAM** (4to semestre), enfocado en la intersección entre el modelado matemático y la neurotecnología.

## 🚀 Objetivo
Mi meta a corto plazo es realizar un doctorado en el **Okinawa Institute of Science and Technology (OIST)**, investigando dinámicas de poblaciones neuronales. A largo plazo, busco fundar una startup que democratice el acceso a la neurotecnología.

## 🛠️ Tech Stack & Workflow
- **Languages:** Python (NumPy, Pandas, SciPy, Matplotlib), HTML/CSS/JS.
- **Environment:** EndeavourOS + Hyprland + Neovim/VS Code.
- **Tools:** Git, GitHub, Notion API, Gemini API.
- **Interests:** Computational Neuroscience, Cognitive Science, BCI.

## 📂 Estructura del Proyecto
Este portafolio está diseñado siguiendo la estética **Catppuccin Mocha** y una interfaz de línea de comandos (CLI).

```text
.
├── index.html          # Estructura principal
├── css/
│   └── style.css       # Estilos (Mocha Palette + Flexbox)
├── js/
│   └── main.js        # Lógica de iconos y animaciones
└── assets/             # GIFs de simulaciones y recursos visuales
```
